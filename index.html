Load your collection

Marvel Snap keeps your collection in a hidden Windows folder. Skip the hunt: copy the path below, hit Sync, and paste it into the file picker's File name box, then Open.

%USERPROFILE%\AppData\LocalLow\Second Dinner\SNAP\Standalone\States\nvprod\CollectionState.json

%USERPROFILE% fills in your username automatically. The file is read in your browser and saved on this device — nothing is uploaded. After the first time, Sync is one click.
